# Python Calculator

A simple calculator program built using Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly terminal input

## How to Run

1. Install Python
2. Download the file
3. Run the program using:

python calculator.py

## Example

Enter first number: 5  
Enter second number: 6  
Choose operation (+, -, *, /): +  

Result: 11
